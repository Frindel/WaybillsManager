Клучевые слова для вывода путевки:
[Number] - номер
[Date] - дата (день, месяц (буквенный), год)
[ShortYear] - две цифры года
[Year] - год
[ChartsMonth] - месяц (буквенный)
[NumberMonth] - месяц
[Day] - день
[CarMap] - модель машины
[StateNumber] - государственный номер
[Driver] - ФИО водителя
[IdentityCard] - удостоверение водителя
[ToRoute] - исходящий маршрут
[BackRoute] - обратный маршрут

Клучевые слова для вывода отчета:
[Numbers] - ячейка начала вывода номеров
[Dates] - ячейка начала вывода даты
[CarMaps] - ячейка начала вывода моделей машин
[StateNumbers] - ячейка начала вывода государственных номеров
[Drivers] - ячейка начала вывода ФИО водителей
[IdentityCards] - ячейка начала вывода удостоверений
[Routes] - ячейка начала вывода маршрутов
